# Vocab Master - 云端生词本

一个智能的浏览器扩展，帮助您自动高亮网页上的生词，统计单词出现频率，并提供多端同步功能。

## 功能特点

- ✨ **自动高亮生词** - 自动扫描并高亮显示网页中您保存的生词
- 📊 **词频统计** - 统计每个生词在不同网页中的出现次数
- 🔍 **悬浮提示** - 鼠标悬停在生词上显示翻译和词性
- ➕ **快速添加** - 右键选中文本即可快速添加到生词本
- 🌐 **自动翻译** - 使用 Google Translate API 获取单词翻译和词性
- ☁️ **云端同步** - 使用 Chrome Storage API 实现多设备同步
- 📝 **词库管理** - 在弹出窗口中查看、管理和删除生词
- 🎨 **优雅设计** - 简洁的用户界面和不干扰的阅读体验

## 安装方法

1. 下载或克隆此项目到本地
2. 打开 Chrome 浏览器，进入扩展程序管理页面 (`chrome://extensions/`)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 扩展安装完成，即可开始使用

## 使用方法

### 添加生词
- 在网页上选中任意英文单词
- 右键点击选择"将 'xxx' 添加到生词本"
- 单词会自动翻译并添加到您的词库中

### 查看生词信息
- 生词会在网页中自动高亮显示（黄色背景）
- 将鼠标悬停在高亮单词上查看翻译和统计信息
- 点击扩展图标查看完整词库列表

### 管理词库
- 点击扩展图标打开词库管理界面
- 查看所有保存的生词，按出现频率排序
- 点击"×"删除单个单词
- 点击"清空全部"重置词库

## 技术实现

- **前端技术**: 原生 JavaScript, HTML5, CSS3
- **扩展标准**: Chrome Extension Manifest V3
- **翻译服务**: Google Translate API
- **数据存储**: Chrome Storage Sync API
- **DOM 操作**: TreeWalker API 高效遍历页面文本

## 项目结构

```
vocab-master/
├── manifest.json      # 扩展配置文件
├── content.js         # 内容脚本，负责页面高亮和交互
├── background.js       # 后台脚本，处理右键菜单和翻译
├── popup.html         # 弹出窗口界面
├── popup.js           # 弹出窗口逻辑
├── style.css          # 高亮样式和提示框样式
└── icon.png           # 扩展图标
```

## 主要文件说明

- **manifest.json**: 定义扩展的基本信息、权限和脚本加载配置
- **content.js**: 页面内容处理，扫描文本、高亮生词、显示悬浮提示
- **background.js**: 右键菜单管理、翻译API调用、消息通信
- **popup.js**: 词库管理界面逻辑，加载和操作存储的单词数据
- **style.css**: 定义高亮样式和工具提示框的视觉效果
